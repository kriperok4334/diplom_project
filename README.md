# diplom_project

Проект дипломной работы по анализу состояния сетевого оборудования на основе данных мониторинга.

## Цель
Разработка интеллектуальной модели, которая получает телеметрию из Zabbix/NOC, анализирует состояние сетевых интерфейсов и выдает заключение с рекомендациями.

## Структура
- `data/` — данные
- `notebooks/` — исследовательские ноутбуки
- `src/` — основной код проекта
- `configs/` — конфигурации
- `artifacts/` — артефакты моделей
- `logs/` — логи
